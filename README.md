# Raint v1.0.0

> A simple almost 1000-line program yet veristile CLI painting program made in rust

Raint aims to be a painting program in a terminal

a few notes:
- This is a personal project , you can explore around if you want
- this was not meant to be handled by companies or handle big stuff

## How to install

### Linux(Any distro) /MacOS (atleast 10.0)

To install raint , you have to do the following:
- Open the release section in https://github.com/mintybrackettemp-hub/raint/
- Download the binary
- in the terminal , type `cd ~/Downloads`, depending in where you downloaded it
- then type `./raint` if you haven't changed the name yet
- Congratulations , now make your masterpiece!

### Windows

This is a different process, don't worry it's not much different

- Install WSL(Ubuntu recomended for this)
- follow the same steps as in the Linux section in the WSL env

## How to use

Usually , just press the `H` key to get help from the bulitin help, No easter eggs , just pure guide

## Requirements

Your terminal needs to be atleast:
- Supporting Unicode
- Supporting Foreground and Background colors in ANSI
- Atleast big enough to fit the program (recomended minimum: 300x300)

### Best terminals for this:

- `kitty`
- Black box from your app store (ensuring you have flatpak)
- ... so many more that support theese
